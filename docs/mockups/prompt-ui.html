How to read this

Variants aren't mutually exclusive across sections — e.g. Checklist reveal (1A) + Suggested glow (2A) + Chat-with-rubric (3A) compose into one coherent flow. Within a section, pick one to build first; nothing prevents adding a second as a setting later.
